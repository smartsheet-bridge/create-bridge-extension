Introduction to Bridge

A Bridge extension is an extension that consists of some very specific functionality to extend Bridge by Smartsheet.

Prerequisite#

Bridge extensions are built on top of the extension core concepts so this documentation assumes you are familiar with the main concepts.

Project Structure#

Along with index.js and package.json needed for all extensions the only extra file necessary for Bridge is the extension.json file. This static configuration file is often referred to as the "Spec File" or "Spec File" and is used to define the extension to the Bridge platform.
