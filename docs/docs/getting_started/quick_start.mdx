## Create Extension

:::caution

This page is incomplete! Thank you for your patience as the team works hard to improve extension development documentation.

:::

:::note

To start developing locally you will need [`NodeJS`](https://nodejs.org/en/) installed on your development machine along with either `npm` or `yarn` and good knowledge of developing `JavaScript` or `TypeScript` applications and services.

:::

We are working hard on an incredibly easy way to create a skeleton extension project. Until then, please refer to the [Manual Setup](/advanced/setup) page or if you are using GitHub feel free to use the ready-made [JavaScript Template](https://github.com/smartsheet-bridge/extension-template-js).
