Quick Start

Create Extension#

note

To start developing locally you will need NodeJS installed on your development machine along with either npm or yarn and good knowledge of developing JavaScript or TypeScript applications and services.

The recommended way to start developing a new Extension is to use the official create-bridge-extension tool along with either npm or yarn to scaffold a new extension project.

  • JavaScript
  • TypeScript

Use npm

npx @smartsheet-bridge/create-extension hello_world

or yarn

yarn create @smartsheet-bridge/extension hello_world

Use npm

npx @smartsheet-bridge/create-extension hello_world --template=ts

or yarn

yarn create @smartsheet-bridge/extension hello_world --template=ts

The tool will scaffold a very basic, but complete, extension project in the folder hello_world. This extension is ready to be deployed.

Deploy Extension#

note

To deploy an extension on Bridge by Smartsheet you will need an account.

To deploy an extension created with the create-bridge-extension tool simply run the following command from the project directory where <insert url here> is the url of your Bridge by Smartsheet account that you can find in the browser when viewing the application, and <insert key here> is an API key that can be generated in your Bridge by Smartsheet account.

  • npm
  • Yarn
npm run deploy --url=<insert url here> --key=<insert key here>
yarn run deploy --url=<insert url here> --key=<insert key here>
